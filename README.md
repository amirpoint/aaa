
## Description

A Nest.js [Rest API](https://www.postman.com/orbital-module-saganist-96879429/workspace/public/documentation/31730966-b25f4e72-1ef8-4f9b-a3d5-425ac5642f2c) for task management system with admin system

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
